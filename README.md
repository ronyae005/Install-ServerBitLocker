# Install-ServerBitLocker
This repository contains a high-performance PowerShell automation suite designed to standardize and accelerate the deployment of BitLocker Drive Encryption across Windows Server environments (2019 and newer).
In enterprise staging or data center environments, manual disk encryption is often time-consuming and prone to human error. This script creates a "self-healing" workflow that verifies hardware prerequisites, installs necessary features, and secures both OS and Data volumes with modern encryption standards.
